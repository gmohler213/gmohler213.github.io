Markovian Tweets

This program takes in text and generates a tweet using markovian text generation. Each word is picked based on the likelihood that it follows the previous word in the input text. For example, given the input "Hi Scott, say hi to Dave for me and say hi to Lisa for me!" the word generated after "hi" would have a 1/3 chance of being "Scott", and a 2/3 chance of being "to".


Example output using the U.S. Constituiton as input:
"Meeting shall be held to Service for a Term of ten Years, in such imminent Danger as will not admit of delay."

Minesweeper

Play a game of my command-line based Minesweeper! Enter a width, height, and number of mines to create a game. Type in an [x, y] location to pick a square, but be careful. Land on just one bomb and you lose!

Gee Mail

This project simulates a private e-mail server. First, either sign-in or register a new account with securely hashed credentials. Next, read or write an encrypted message from/to another user on the server using a custom built encryption scheme.
